# ETL - Sistema de Alarmes de Formulação

Pipeline ETL para processamento de logs de alarmes do sistema de formulação farmacêutica.

## Funcionalidades
- Extração de logs de múltiplos arquivos
- Transformação e normalização de dados
- Carga em banco SQLite com controle de versão
- Suporte a encoding múltiplo
- Logging detalhado de operações

## Requisitos
- Python 3.9+
- SQLite 3

## Instalação
```bash
pip install -r requirements.txt
```
